# Changelog

## v1.0.0 - 2026-06-29

- Initial public release.
- Adds the `availability_groupname` Moodle availability restriction.
- Supports exact, contains, and starts-with matching against course group names.
- Stores one group-name rule per availability condition for straightforward reporting.
- Tested with Moodle 4.5.11+ and Moodle 5.2.1+.
